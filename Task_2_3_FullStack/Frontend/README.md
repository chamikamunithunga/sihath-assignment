# Urban Harvest Hub

A community platform for urban gardeners to share produce, knowledge, and resources.

## Features
- Browse organic produce and plants
- Book workshops
- Local weather updates
- Shopping cart functionality

## Tech Stack
- React
- Vite
- Tailwind CSS

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

## Structure
- `/public/data`: Seed data for the application
- `/src/components`: Reusable UI components
- `/src/pages`: Route components
- `/src/contexts`: Global state management
