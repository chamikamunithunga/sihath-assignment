HOW TO ADD IMAGES FOR YOUR WEBSITE
====================================

This folder (public/Images/) contains ALL images used on your website.
All images use simple paths like: /Images/filename.jpg

REQUIRED IMAGES - Place these files in this folder:
----------------------------------------------------

HOMEPAGE:
1. hero-banner.jpg         - Main banner on homepage

CATEGORY CARDS:
2. products.jpg            - Products category card
3. workshops.jpg           - Workshops category card  
4. events.jpg              - Events category card

INDIVIDUAL WORKSHOPS:
5. workshop-gardening.jpg     - Urban Gardening Workshop
6. workshop-photography.jpg   - Eco Photography Class
7. workshop-journaling.jpg    - Green Writing & Journaling
8. workshop-diy.jpg           - DIY Natural Products

INDIVIDUAL PRODUCTS:
9. product-veggie-box.jpg     - Organic Vegetable Box
10. product-water-bottle.jpg  - Eco-Friendly Water Bottle

INDIVIDUAL EVENTS:
11. event-summit.jpg          - Sustainability Summit 2024

IMPORTANT:
----------
- Images MUST be named EXACTLY as shown above
- Use .jpg, .png, or .webp format
- Recommended size: 800x600 pixels or larger
- Keep file sizes under 500KB for fast loading

Where to find free images:
- Unsplash.com (free high-quality photos)
- Pexels.com (free stock photos)
- Pixabay.com (free images)

Just download the images you like, rename them to match the list above, 
and place them in this folder (public/Images/)!
